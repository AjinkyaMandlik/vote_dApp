## ğŸ—ï¸ Smart Contract

The poll contract is written in Rust using the Soroban SDK and deployed to Stellar testnet.

**Contract functions:**
- `vote_yes(voter: Address)` â€” records a YES vote
- `vote_no(voter: Address)` â€” records a NO vote

  --source YOUR_ACCOUNT \
  --network testnet
